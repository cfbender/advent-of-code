### Dependencies
    - Deno
    - Rust (Rustup/Cargo)
    - Elixir (Mix/IEX)
    
### Running

##### Typescript
In respective day directory:
```bash
$ deno run --allow-read ./index.ts
```

##### Elixir (2020 only)
In respective day directory:
```bash
$ iex -S mix 
iex > Aoc.main()
```

##### Rust
In respective day directory:
```bash
$ cargo run
```
